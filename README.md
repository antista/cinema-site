# Сайт кинотеатра "Галактика"

## Описание
Сайт кинотеатра "Галактика" с возможностью покупки билетов.

### Запуск сервера
* `set FLASK_APP=cinema.wsgi` on Windows
* `export FLASK_APP=cinema.wsgi` on Linux
* `flask run`

### Запуск тестов
* `python -m pytest`

### Запуск линтера
* `flake8 cinema/`
